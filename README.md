# Kane
 Simple language for doing math quickly and exporting calculation data

# Language

Finding the difference:
```py
# Input

x = difference of 4 and 2
write(x)

# Output
2
```
<br>

Finding the sum:
```py
# Input
x = add 5 and 10
write(x)

# Output
15
```
<br>

Check if is rational:
```py
# Input
number = sqrt of 3
x = is number rational
write(x)

# Output
False

```
